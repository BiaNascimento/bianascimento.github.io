About Eternity

Responsabilities: Game Designer and Level Designer.

Check the game

Eternity is a game made for Global Game Jam 2021, which theme was Lost & Found. The game is a top-down platformer where you control Ione, a young climber that finds herself lost in a mysterious cave and must find her way out.

I was responsible for the level and game design, where 3 levels were designed for the game jam, with NPC encounters and mini puzzles that the player must solve to leave the cave.

The major difficult was to deal with time management during an online game jam, with a team that I mostly didn’t know. Due to the lack of communication with the programming member, there were more features designed than would be possible to implement.

We tried to overcome the challenge by implementing a priority system for the future implementations. It allowed us to deliver a functional game, but with a lot of features missing.

Design Process

Brainstorming

there was a long online brainstorming session with the team, where a lot of ideas appeared, and we chose the most viable one to make in 2 days.

Prototypes

as we had time management problems we had few functional prototypes before the deadline, but we engaged to validate technical aspects of the game.

Playtest

we did a few internal playtests to attest the game functionability.

Final version

The game was released with the main actions and there are plans for future updates.
